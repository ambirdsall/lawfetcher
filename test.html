RULES

In the enumberated abbreviations ('U.S.C.', 'stat.') use of periods and spaces is optional.
So far as the code is concerned, 'U S C' == 'USC' == 'U.S.C' == 'U. S. C.'
Rules are applied in order, and citations are classified by the first set of rules that match.
  1. Contains 'U.S. Const'
  2. Contains 'C.F.R.' OR 'U.S.C.'
  3. Begins with 'F' AND ('ed. ' OR 'ederal ') OR optional '.'; 'R' AND ('ule' OR 'ules') OR optional '.'
  4. Contains 'U.S.' OR 'S. Ct.' OR 'F.' optional ' Supp.' optional '{digit}d'
  5. Contains 'Const'
  6. Contains 'law' OR 'laws' OR 'stat.' OR 'code' OR 'regs.' OR 'rule'
  7. All others
