A few tips on styling

  1. Bootstrap's stylesheet is loaded in the <head>; it has a relatively thorough set of defaults. If all you intend to add is a few <p> tags and a header or two, the prewritten styles should be entirely sufficient.
  2. A possible exception: add the class 'serif' to any HTML element to display its text in Century Schoolbook. For example, the contents of this <span> tag and the <h3> above.
  3. Should you wish to delve more deeply into the styling, quite a few other useful classes are already defined. You may find Bootstrap's documentation useful.
